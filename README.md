# landing — Live Shop Pro

Landing page de venda do **Live Shop Pro**, seguindo a identidade visual de **qyvolive.com.br**:

- Paleta dark roxa (`#0E0B14` → `#3D3148`) com accent magenta `#FF1F8E`
- Tipografia: **Fraunces** (display, italic), **Inter** (body), **JetBrains Mono** (eyebrow/tag)
- Hook gradient como signature visual do produto

## Como ver localmente

Abre `index.html` direto no Chrome (duplo clique). Funciona offline (fontes do Google CDN).

## Como subir pra qyvolive.com.br/live-shop-pro/

1. Copia tudo dessa pasta pro repo do site da Qyvo Live na pasta `/live-shop-pro/`
2. Commit + push
3. Acessa `qyvolive.com.br/live-shop-pro/`
4. Atualiza o card do Live Shop Pro na home da Qyvo: troca "Em breve" / "Notificar quando lançar" por "Disponível" + CTA `/live-shop-pro/`
