SETUP INICIAL · 10 MINUTOS

Instala o cabo virtual
e tá pronto.

O Quick Test gera o áudio. Pra esse áudio chegar no Live Studio (no lugar do seu microfone), precisa de um cabo virtual — programa pequeno que conecta a saída de áudio do Chrome direto na entrada de mic do Live Studio.

Por quê esse passo extra? No Mac e no Windows, sistemas não permitem que uma aba do Chrome envie áudio direto pra outro aplicativo (Live Studio). O cabo virtual resolve isso. Você instala 1 vez, configura 1 vez, e nunca mais mexe. Os clientes que usam Quick Test rodam o cabo virtual em background sem nem perceber.
🍎

Se você usa Mac

GRÁTIS · RECOMENDADO
BlackHole 2ch
a partir deR$ 0

Cabo virtual open-source da Existential Audio. Roda nativo no macOS, aparece como dispositivo de áudio normal. Cobre 100% do uso do Quick Test.

  1. Acessa existential.audio/blackhole e clica em "Download Free"
  2. Preenche email e nome (eles mandam o link de download por email)
  3. Abre o arquivo .pkg que chegou no email e segue o instalador (next, next, instalar, senha do Mac, concluir)
  4. Abre Configurações do Sistema → Som — confere que aparece BlackHole 2ch em "Entrada"
  5. No Quick Test, vai no campo "Saída de áudio" e escolhe BlackHole 2ch
  6. No Live Studio, vai em Configurações → Microfone → escolhe BlackHole 2ch

PAGO · AVANÇADO
Loopback
a partir de~$109 USD

Da Rogue Amoeba, é o cabo virtual top de linha pra Mac. Permite criar múltiplos cabos virtuais, mixar fontes (Chrome + Spotify + microfone real, por exemplo), painel visual de controle. Versão grátis funciona com limite de 20min.

  1. Acessa rogueamoeba.com/loopback
  2. Baixa a versão demo gratuita (ou compra direto)
  3. Instala e abre o app
  4. Cria um "New Virtual Device", dá um nome (ex: "Quick Test → Live")
  5. Adiciona o Chrome como "Source"
  6. No Quick Test, escolhe esse cabo virtual no campo "Saída"
  7. No Live Studio, escolhe o mesmo cabo como microfone

Quando vale a pena: se você roda Quick Test + Spotify + sons ambiente simultâneo, ou quer setup avançado de mixagem ao vivo. Pra uso normal, BlackHole basta.

Se você usa Windows

GRÁTIS · RECOMENDADO
VB-Cable
a partir deR$ 0

Cabo virtual mais usado no Windows. Donationware (contribuição opcional). Funciona em Windows 10 e 11, cobre 100% do Quick Test sem limitação.

  1. Acessa vb-audio.com/Cable
  2. Baixa o VBCABLE_Driver_Pack (zip)
  3. Extrai o zip em uma pasta qualquer
  4. Clica com botão DIREITO em VBCABLE_Setup_x64.exe → Executar como administrador
  5. Clica em "Install Driver" e reinicia o Windows
  6. Abre Configurações de Som do Windows → confere se aparece CABLE Input e CABLE Output
  7. No Quick Test, escolhe CABLE Input em "Saída de áudio"
  8. No Live Studio, escolhe CABLE Output como microfone

PAGO · MULTI-LIVE
Voicemeeter Potato
a partir de~$30 USD

Da mesma VB-Audio, é uma mesa de mixagem virtual completa. Permite ter vários cabos virtuais simultâneos — essencial pra quem roda múltiplas lives em paralelo (Sandboxie / multi-conta). Versão grátis "Banana" é mais simples.

  1. Acessa vb-audio.com/Voicemeeter/potato.htm
  2. Baixa e instala (clica em "Install" como admin)
  3. Reinicia o Windows
  4. Abre o Voicemeeter — vai aparecer um painel de mixer com várias entradas/saídas
  5. Configura cada "Hardware Input" pra um perfil (Live 1, Live 2, etc)
  6. Cada cabo virtual aparece como dispositivo separado no sistema
  7. No Quick Test (cada instância), escolhe um cabo diferente

Quando vale a pena: se você roda 2+ lives simultâneas (Sandboxie, multi-conta TikTok). Pra 1 live só, VB-Cable basta.
